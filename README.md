# 🎧 Rotinas++

> Aplicativo móvel nativo para Android focado na automação de perfis de áudio baseada em rotinas semanais.

O **Rotinas++** resolve a inconveniência de alterar manualmente o volume do smartphone durante o dia a dia. Com ele, estudantes e trabalhadores podem agendar em quais horários e dias da semana o celular deve entrar automaticamente no modo Silencioso/Vibrar (momentos de foco) e quando deve retornar ao Som Padrão (momentos de pausa).
